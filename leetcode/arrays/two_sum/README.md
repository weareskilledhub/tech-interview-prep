# Two Sum

## Problem
Given an array of integers `nums` and an integer `target`, return indices of the two numbers such that they add up to `target`.

## Example
Input:
nums = [2, 7, 11, 15]
target = 9

Output:
[0, 1]

## Approach
Use a hash map to store previously seen numbers and their indices.

## Time Complexity
O(n)

## Space Complexity
O(n)
